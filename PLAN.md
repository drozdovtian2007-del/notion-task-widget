# План реализации: Виджет для быстрого ввода задач в Notion

## 📋 Описание проекта
Создать iOS виджет на рабочем экране, который позволяет быстро вводить текст задачи → автоматически добавляет в Notion базу данных с датой/временем и чекбоксом статуса.

---

## 💰 Стоимость
**Полностью бесплатно** (только используем уже имеющиеся подписки)

---

## 🏗️ Архитектура

```
iPhone (iOS Shortcut Widget)
        ↓
   [Ввод текста]
        ↓
  Notion API
        ↓
  База данных "Задачи"
```

### Компоненты:
1. **iOS Shortcut** — виджет на рабочем экране (бесплатно)
2. **Notion API Integration** — подключение к твоему Notion (бесплатно)
3. **Notion База "Задачи"** — создаём заново со структурой

---

## 📊 Структура Notion базы "Задачи"

Создадим базу с такими полями:

| Поле | Тип | Описание |
|------|-----|---------|
| **Задача** | Text (название) | Текст задачи |
| **Статус** | Checkbox | ☑️ выполнено / ☐ не выполнено |
| **Дата создания** | Date | Автоматически записывается |
| **Время создания** | Text | Автоматически записывается |

---

## 🚀 Этапы реализации

### Этап 1: Подготовка (5 мин)
- [ ] Ты даёшь доступ к Notion (создаём integration token)
- [ ] Я создаю в твоём Notion базу данных "Задачи"

### Этап 2: Создание iOS Shortcut (15 мин)
- [ ] Создаю Shortcut на iPhone который:
  - Отображает окно ввода текста
  - Отправляет в Notion API
  - Добавляет дату/время автоматически
  - Показывает уведомление "Задача добавлена"
- [ ] Ты устанавливаешь Shortcut на iPhone
- [ ] Создаёшь виджет на рабочем экране

### Этап 3: Тестирование (5 мин)
- [ ] Вводишь первую задачу через виджет
- [ ] Проверяешь что она появилась в Notion
- [ ] Тестируешь чекбокс (отмечаешь выполненные)
